## 🧠 About Us

**3AMI Team** is a collaborative software team formed by Computer Science students at the **Faculty of Science, Suez Canal University**. We created this organization as a shared space to build, publish, and maintain every software project we work on together.

Our work spans the full technology spectrum — from mobile & desktop apps to full-stack web development, data science pipelines, and backend systems — driven by a shared passion for clean code and real-world impact.

> _Every project here was built together. Every commit reflects a team effort._

## 📌 How We Work

```
┌─────────────────────────────────────────────────────────────┐
│                    OUR COLLABORATION FLOW                    │
│                                                             │
│   Idea  ──▶  Design (Figma)  ──▶  Architecture Decision    │
│                                           │                 │
│                                           ▼                 │
│   Deploy  ◀──  Review & Merge  ◀──  Feature Branches       │
│                                                             │
│   Every project lives here — open, versioned, documented.  │
└─────────────────────────────────────────────────────────────┘
```

- 🌿 **Branch per feature** — no direct pushes to `main`
- 🔍 **Code review** before every merge
- 📋 **Trello** for task tracking & sprint planning
- 📐 **Figma** for UI/UX design before implementation
- 📄 **Every repo has a README** — always documented
